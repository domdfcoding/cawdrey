=============
Base Classes
=============

About
========

``FrozenBase`` is the base class for :class:`~cawdrey._frozendict.frozendict`
and :class:`~cawdrey.frozenordereddict.FrozenOrderedDict`.
If you wish to construct your own frozen dictionary classes,
you may inherit from this class.
